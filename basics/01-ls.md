# ls Command

## What is `ls`?

The `ls` command lists the files and directories in the current working directory.

---

## Syntax

```bash
ls [OPTIONS] [PATH]
```

---

## Basic Usage

```bash
ls
```

Lists all visible files and directories.

---

## Useful Options

### Long Listing

```bash
ls -l
```

Displays file permissions, owner, size, and modification date.

### Show Hidden Files

```bash
ls -a
```

Displays hidden files (those starting with a `.`).

### Human-Readable File Sizes

```bash
ls -lh
```

Shows file sizes in KB, MB, or GB.

---

## Example

```bash
mkdir Linux
cd Linux
touch notes.txt
ls -l
```

Output:

```text
-rw-r--r-- 1 user user 0 Jun 28 notes.txt
```

---

## Real-World Use

Before editing or deleting files, use `ls` to verify the contents of the directory.

---

## Common Mistakes

❌ `Is`

✅ `ls`

Linux commands are case-sensitive.
