# cd Command

## What is `cd`?

The `cd` (**Change Directory**) command is used to navigate between directories in the Linux file system. It allows you to move from one directory to another.

---

## Syntax

```bash
cd [DIRECTORY]
```

---

## Basic Usage

### Move to a Directory

```bash
cd Documents
```

Changes the current working directory to `Documents`.

---

## Common Usage

### Go to the Home Directory

```bash
cd
```

or

```bash
cd ~
```

Moves you to your home directory.

---

### Go to the Parent Directory

```bash
cd ..
```

Moves one directory up from the current directory.

---

### Go to the Previous Directory

```bash
cd -
```

Switches back to the previous working directory.

---

### Go to the Root Directory

```bash
cd /
```

Moves to the root (`/`) directory of the Linux file system.

---

### Navigate Using an Absolute Path

```bash
cd /home/vishwa/Documents
```

Moves directly to the specified directory.

---

### Navigate Using a Relative Path

```bash
cd Projects/Linux
```

Moves to the `Linux` directory inside the `Projects` directory from your current location.

---

## Example

```bash
pwd

mkdir Linux

cd Linux

pwd
```

Output:

```text
/home/vishwa
/home/vishwa/Linux
```

---

## Real-World Use

When working on projects, you'll frequently switch between directories.

Example:

```bash
cd /var/www/html
```

Moves to the web server's document root.

Or:

```bash
cd ~/Projects/MyApp
```

Moves to your project directory before editing files or running applications.

---

## Common Mistakes

### Forgetting Spaces

❌ Incorrect

```bash
cd..
```

✅ Correct

```bash
cd ..
```

---

### Navigating to a Non-Existing Directory

❌

```bash
cd MyProject
```

Output:

```text
bash: cd: MyProject: No such file or directory
```

Always verify the directory exists:

```bash
ls
```

---

### Using the Wrong Path

If you're unsure where you are, first check your current location.

```bash
pwd
```

Then navigate to the desired directory.

---

## Tips

* Use `pwd` to verify your current directory.
* Use `ls` to view available files and directories before navigating.
* Press the **Tab** key to auto-complete directory names.

Example:

```bash
cd Doc<Tab>
```

Automatically completes to:

```bash
cd Documents
```

(if only one matching directory exists).

---

## Interview Question

**Q:** What is the purpose of the `cd` command?

**A:** The `cd` command is used to change the current working directory in Linux.

---

**Q:** What is the difference between `cd ..` and `cd /`?

**A:**

* `cd ..` moves one directory up (to the parent directory).
* `cd /` moves directly to the root directory.

---

## Quick Summary

| Command                 | Description                      |
| ----------------------- | -------------------------------- |
| `cd`                    | Go to the home directory         |
| `cd ~`                  | Go to the home directory         |
| `cd ..`                 | Move to the parent directory     |
| `cd -`                  | Return to the previous directory |
| `cd /`                  | Go to the root directory         |
| `cd directory_name`     | Move to a specific directory     |
| `cd /path/to/directory` | Move using an absolute path      |
